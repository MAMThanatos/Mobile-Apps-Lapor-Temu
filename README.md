# Mobile Apps - Lapor Temu

Repositori ini dibuat khusus untuk pengumpulan file APK aplikasi **Lapor Temu** sesuai dengan instruksi tugas yang diberikan. Repositori ini tidak memuat *source code* proyek.

## Tentang Aplikasi
**Lapor Temu** adalah aplikasi berbasis mobile yang dirancang untuk memudahkan pengguna dalam melaporkan atau menemukan barang-barang yang hilang secara cepat dan efisien.

---

## Cara Mengunduh Aplikasi (APK)
Anda dapat mengunduh file aplikasi `.apk` siap pakai melalui 2 cara di bawah ini:

### Cara 1: Unduh Langsung dari Halaman Utama
1. Klik file **`LaporTemu.apk`** atau **`LaporTemuAPK.zip`** yang ada di daftar file di atas.
2. Klik tombol **Download** (ikon unduh) di pojok kanan atas area file tersebut.

### Cara 2: Unduh Melalui Menu Releases (Rekomendasi)
1. Lihat ke bagian kanan halaman ini, lalu klik menu **Releases** (**Aplikasi Lapor Temu APK**).
2. Di bagian **Assets**, klik file **`LaporTemu.apk`** untuk langsung mengunduh versi terbaru yang stabil.
